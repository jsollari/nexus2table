This distribution includes material for the nexus2table computer program

The directory structure of this archive is as follows
+-base directory
	+--doc
	+--examples
	+--src

The base directory contains the following files:

README.txt  - this file
LICENSE.txt - GNU license v3 for the nexus2table software
Makefile - makefile to compile nexus2table executables

The contents of the subdirectories are as follows:

doc - this directory contains user guide for nexus2table

examples - this directory contains example files to use with nexus2table and to go through the quick-start guides

src - this directory contains the source code of the nexus2table package
